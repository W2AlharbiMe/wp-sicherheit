### Wordpress Sicherheit

<p>a security plugin made for wordpress.</p>

### What does it include?

- Disable Shortlinks
- Disable Pingback tag
- Disable wp head:

```
- wp_resource_hints
- rsd_link
- wlwmanifest_link
- wp_generator
- wp_shortlink_wp_head
- feed_links
- feed_links_extra
- index_rel_link
- start_post_rel_link
- parent_post_rel_link
- adjacent_posts_rel_link_wp_head
```
